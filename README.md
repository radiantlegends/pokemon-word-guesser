# Introduction
This is a project created based on ZaneGames's video:\
[Guess The Pokemon But FEWEST Hints Wins!](https://youtu.be/10x-S7t1Tq0)

# The Rules
Each round, there is one hint giver and any number of guessers. The hint giver's role is to use as little hints as possible to get the other players to guess their list of prompts. Each guesser gets a point per correct guess. At the end of the round, the hint giver is scored based on how many hints below 25 they gave. Going over 25 results in negative points.
